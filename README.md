## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/atul-singh-1/Python-Calculator.git
```

Go to the project folder:

```bash
cd Python-Calculator
```

Run the program:

```bash
python calculator.py
```

If `python` doesn't work on your system, try:

```bash
py calculator.py
```
